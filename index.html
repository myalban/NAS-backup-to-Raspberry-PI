NAS backup to the Raspberry PI

Shell script to backup some NAS folder mounted as CIFS to the Raspberry pi.

What I need ?

This script can run with a lot configuration but here is my setup :

  • NAS with mounted folder as CIFS to share it around Windows computers
  • Raspberry PI type B
  • USB HD, 500G
  • HUB USB2 with PSU

How does it works ?

It's verry simple, the script mount the USB drive and the NAS CIFS, then rsync needed folder. After that, it unmount drive and stop

How to run it periodically ?

Use the crontab !

SSH your RPI, then do this :

sudo su root crontab -e

Then add this line :

0 9 * * * /home/pi/maintenance/backup_nas.sh>> /home/pi/maintenance/cron.log 2>&1

This will execute the shell script /home/pi/maintenance/backup_nas.sh each days at 9h00 AM, then output logs on this file : /home/pi/maintenance/cron.log
